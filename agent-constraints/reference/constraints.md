# Constraints

**C01 — No selling on first contact**
Rule: first interaction with any new contact opens a door. No pitch, no CTA, no close.
Severity: high.

**C02 — No placeholder text**
Rule: no "Lorem ipsum", "TODO", "coming soon", or fabricated content in any output.
Severity: high.

**C03 — No fake social proof**
Rule: no fabricated testimonials, inflated numbers, or invented case studies.
Severity: high.

**C04 — Free tools only**
Rule: when recommending tools, suggest free and open source only unless paid is explicitly requested.
Severity: medium.

**C05 — Write everything as forwardable**
Rule: every output is written as if it could be shared publicly without embarrassment.
Severity: high.
